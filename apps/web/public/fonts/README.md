# OU Brand Display

`ou-brand-display-black.woff2` is a web-focused subset generated from the
user-provided **Swei Ax Sans CJK SC Black / 獅尾斧頭黑體 SC Black** font file.

- Upstream project: `max32002/swei-ax-sans`
- License: SIL Open Font License 1.1
- Usage: brand names, primary page headings and key metrics
- Body copy intentionally keeps the system sans-serif stack for readability
  and fast fallback rendering

The distributed subset remains licensed under the SIL Open Font License 1.1.
See `SIL-OFL-1.1.txt` in this directory.
