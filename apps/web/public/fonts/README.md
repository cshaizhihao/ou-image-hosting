# OU Brand Display

`ou-brand-display-black.woff2` is generated from the user-provided **Swei Ax
Sans CJK SC Black / 獅尾斧頭黑體 SC Black** font file and is used for OU-Image
Hosting brand headings, custom homepage copy, login copy and key visual text.

The file is intentionally kept as a full CJK-capable WOFF2 instead of a tiny UI
subset, so custom Chinese headings do not fall back to system fonts after users
edit site copy.

- Upstream project: `max32002/swei-ax-sans`
- License: SIL Open Font License 1.1
- Usage: brand names, primary page headings, custom homepage copy and key metrics
- Body copy intentionally keeps the system sans-serif stack for readability
