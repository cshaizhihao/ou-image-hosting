## [1.0.0] - 2026-07-11

### Added

- 发布 OU-Image Hosting / 欧记图床首个正式版本。
- 提供公共上传首页、公共图库、站内浮窗预览和登录用户上传历史。
- 支持访客直传 / 登录后上传策略、批量公开 / 隐藏和公共上传审计。
- 提供图片库、相册、标签、回收站、分享链接、收藏和批量管理能力。
- 支持站点所有者、普通注册用户、子管理员和 API Token 的清晰权限边界。
- 提供站点名称、副标题、Logo、登录页文案、公共首页文案和主题外观配置。
- 支持本地存储、Amazon S3、Cloudflare R2 和 S3-compatible 存储。
- 提供一键安装脚本、Cloudflare 小黄云模式、Caddy 反向代理与源站证书。
- 提供 `ouih` 运维命令，包含状态、日志、更新、诊断、备份、恢复、回滚和卸载。

### Changed

- 将当前稳定能力收口并重新定位为正式版 `v1.0.0`。
- README 改写为面向正式作品的介绍，移除构建阶段内容和私密资料痕迹。
- 发布说明收敛为当前正式版本，避免旧阶段版本干扰用户理解。

### Security

- 公共上传支持速率限制、每日额度、可选人机验证和 IP 封禁。
- 上传文件名与公开展示内容会做安全清理。
- 远程 URL 上传会阻断私网、保留地址和 DNS rebinding 风险。
- 后台不会读取或展示用户密码，只支持安全重置流程。

### Tests

- 低并发运行类型检查、单元测试、安装器测试、`ouih` 脚本测试和生产构建。
