# Changelog

本项目遵循 Semantic Versioning。

## [0.3.0] - 2026-07-10

### Added

- 新增三步首次安装向导、Node.js 与数据目录环境检查。
- 新增 JSON 文件持久化仓库，用于站点配置、用户、会话与密码重置记录。
- 新增管理员创建、开放注册开关、登录、注册、退出和会话查询接口。
- 新增 `scrypt` 密码散列、会话令牌摘要、登录失败锁定和接口限流。
- 新增找回密码与一次性密码重置流程，重置后自动撤销全部旧会话。
- 新增密码强度提示、首次使用引导和浅色、深色、跟随系统主题偏好。
- 新增认证路由保护、真实用户菜单与退出操作。
- 新增 6 组 API 集成测试，共覆盖安装、校验、认证、会话、重置与防枚举行为。
- 新增安装、配置、首次引导、已登录工作区和移动端深色登录截图。

### Security

- 会话使用 `HttpOnly`、`SameSite=Lax` Cookie，生产环境默认启用 `Secure`。
- 服务端仅保存会话与密码重置令牌的 SHA-256 摘要。
- 认证响应禁止缓存，敏感请求字段与 Cookie 在日志中自动脱敏。
- 写请求校验来源域，登录与密码重置接口启用独立限流。
- 密码找回响应不暴露账号是否存在。

### Changed

- Web 通过同源 `/api/*` 代理访问 Fastify，避免浏览器端暴露内部 API 地址。
- README 更新 v0.3.0 安装、初始化、认证流程、安全配置与真实截图。
- 十轮路线图推进至第 2 / 10 轮。
