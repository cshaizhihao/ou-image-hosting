# Changelog

本项目遵循 Semantic Versioning。

## [0.2.0] - 2026-07-10

### Added

- 初始化 pnpm Monorepo，包含 Web、API、UI、Shared 与 Config 包。
- 建立 Next.js 应用壳层、Fastify 健康检查和共享导航模型。
- 落地浅色与深色 Design Token、响应式侧栏、顶栏与移动端导航。
- 新增命令面板、通知抽屉、用户菜单、状态页与加载骨架。
- 新增图片拖拽、文件选择和浏览器本地预览。
- 新增 Vitest 测试、Playwright 截图流程与低 CPU 构建脚本。
- 新增桌面端与移动端的浅色、深色真实应用截图。

### Changed

- README 更新为可运行项目说明、安装命令和当前十轮进度。
- 路线图改为从 v0.2.0 到 v1.0.0 的十个实施里程碑。
- 仓库不再保存开发 Prompt 文档。

## [0.1.1] - 2026-07-10

### Changed

- 明确仓库仅保存项目实现、正式设计资产、测试、部署与版本文档。
- 将会话草稿、私人参考资料、AI 提示词和临时分析排除在版本库之外。

- 新增 README、版本文件、许可证和仓库同步规则。
