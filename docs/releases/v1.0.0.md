# OU-Image Hosting v1.0.0

这是 OU-Image Hosting / 欧记图床的首个正式版本。

这个版本把项目定位为一个完整、可长期自托管使用的图片资产管理工具：它不只是上传图片和生成外链，也包含公共图床、后台管理、相册整理、权限边界、品牌自定义、安装器和日常运维能力。

## ✨ 版本亮点

- 🍎 **更适配 iPhone 图片**：支持 HEIC / HEIF 上传，并为部分环境补充转码兜底生成缩略图；公共上传可选接收 Live Photo 的 MOV 动态片段。
- ⚙️ **更清楚的设置中心**：站点设置、站点外观和图片处理拆开；高频上传开关单独放置，外观配置用折叠分类收纳。
- 🐱 **更完整的公共图床**：公共上传首页、公共图库、站内浮窗预览、登录用户上传历史和公开/隐藏控制。
- 🖼️ **更顺手的图片管理**：图片库多选、批量公开/隐藏、批量收藏、加入相册、移入回收站和分享链接。
- 📚 **更像相册的相册**：相册封面、描述、数量、排序、批量补图和批量移出图片。
- 🔐 **更清楚的权限模型**：普通注册用户只能使用公共图床和自己的历史上传，游客上传不会混入管理员个人历史，子管理员只能由站点所有者在后台授权。
- 🎨 **更有温度的品牌体验**：站点名称、副标题、Logo、登录页文案、公共首页文案、主题模式和受控主题色。
- ☁️ **更友好的存储配置**：本地存储、Amazon S3、Cloudflare R2 和 S3-compatible 配置向导与连接测试。
- 🧑‍🔧 **更省心的安装与运维**：一键安装、Cloudflare 小黄云、Caddy 反向代理、源站证书和 `ouih doctor`。

## 🚀 安装

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```

安装完成后，使用安装器输出的访问地址进入站点，创建第一个管理员账号即可开始使用。

## 🧰 运维

```bash
ouih status
ouih logs
ouih update
ouih doctor
ouih backup
ouih restore
ouih rollback
```

## 内测修复补充

- 🎨 站点主题色、公共上传页文案、登录页文案在无缓存时使用中性加载态，避免刷新时先闪回默认珊瑚粉和默认文案。
- 📸 Live Photo 上传支持 JPEG / HEIC / HEIF 封面图与同名 MOV / MP4 动态片段自动配对；单独一张 JPEG 无法恢复动态效果。
- 🧺 登录用户的“我的上传历史”支持本页全选、批量删除到回收站，并固定为 12 张一页。
- 🧩 站点外观可自定义公共首页三枚欢迎卡片的预置图标、标题和说明。
- 🧮 公共上传页的人机验证题目会在访客上传可用后自动拉取，避免显示“暂时无法获取题目”。
- 🔤 品牌标题字体改为完整加载用户提供的獅尾斧頭黑體，避免自定义中文大标题混用系统字体，并减少首屏字形跳变。
- ✨ 微调品牌大标题字距与行高，让首页、登录页和上传页的标题更松弛、更好读。
- 🌙 公共首页三张欢迎卡片改为“温柔欢迎款”文案，并加入轻量星点呼吸动效，让首页更有一点陪伴感。
- 🪄 站点外观里的首页欢迎卡片编辑器修复窄宽度重叠，并把图标选择改成带预览的小浮窗。
- 🪟 欢迎卡片图标选择改为 Portal 弹窗，避免被折叠面板遮盖；主题色补齐选中重点边框。
- 🧭 后台切页时保留真实侧栏，只显示约 1.5 秒的环形与进度动画；从相册打开图片后可返回原相册。
