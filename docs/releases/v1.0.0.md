# OU-Image Hosting v1.0.0

这是 OU-Image Hosting / 欧记图床的首个正式版本。

这个版本把项目定位为一个完整、可长期自托管使用的图片资产管理工具：它不只是上传图片和生成外链，也包含公共图床、后台管理、相册整理、权限边界、品牌自定义、安装器和日常运维能力。

## ✨ 版本亮点

- 🍎 **更适配 iPhone 图片**：支持 HEIC / HEIF 上传，并为部分环境补充转码兜底生成缩略图。
- 🐱 **更完整的公共图床**：公共上传首页、公共图库、站内浮窗预览、登录用户上传历史和公开/隐藏控制。
- 🖼️ **更顺手的图片管理**：图片库多选、批量公开/隐藏、批量收藏、加入相册、移入回收站和分享链接。
- 📚 **更像相册的相册**：相册封面、描述、数量、排序、批量补图和批量移出图片。
- 🔐 **更清楚的权限模型**：普通注册用户只能使用公共图床和自己的历史上传，子管理员只能由站点所有者在后台授权。
- 🎨 **更有温度的品牌体验**：站点名称、副标题、Logo、登录页文案、公共首页文案、主题模式和受控主题色。
- ☁️ **更友好的存储配置**：本地存储、Amazon S3、Cloudflare R2 和 S3-compatible 配置向导与连接测试。
- 🧑‍🔧 **更省心的安装与运维**：一键安装、Cloudflare 小黄云、Caddy 反向代理、源站证书和 `ouih doctor`。

## 🚀 安装

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```

安装完成后，使用安装器输出的访问地址进入站点，创建第一个管理员账号即可开始使用。

## 🧰 运维

```bash
ouih status
ouih logs
ouih update
ouih doctor
ouih backup
ouih restore
ouih rollback
```
