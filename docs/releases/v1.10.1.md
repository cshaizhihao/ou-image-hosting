# OU-Image Hosting v1.10.1

这是一个登录与工作区初始化热修复版本。

## 修复内容

- 修复浏览器残留旧 `ou-workspace-id` 时，登录后进入 onboarding 页面但“进入工作区”按钮不可用的问题。
- `/auth/session` 会在旧工作区失效时回落到当前账号的默认工作区，避免用户被重新踢回登录页。
- 登录和注册成功后会清理旧工作区缓存，onboarding 再写入当前有效工作区。

## 验证

- API 回归测试覆盖旧工作区回落场景。
- Web 类型检查覆盖登录、注册、onboarding 和通用 API 请求封装。

## 升级建议

已安装用户可运行：

```bash
ouih update
```

如果浏览器仍保留旧页面状态，升级后刷新一次登录页即可。
