# OU-Image Hosting v1.5.0

`v1.5.0` 完成第 6 轮：公共上传权限与公共图库预览体验升级。

## 新增内容

- 后台“设置中心 → 站点与处理”新增“上传需要登录”开关。
- 管理员可以选择公共页允许访客直接上传，或要求登录后才能上传。
- 登录用户从公共页上传时，图片归属到自己的默认工作区，便于后续继续管理。
- 公共图库缩略图点击后改为站内浮窗预览，不再跳转到新网页。

## 交互优化

- 预览浮窗支持：
  - 鼠标滚轮切换上一张/下一张。
  - 键盘 `←` / `→` 切换图片。
  - 移动端滑动切换图片。
  - 左右按钮和关闭按钮。
- 上传权限不足时，公共页直接提示登录，不再让用户走到失败才知道原因。
