# OU-Image Hosting v1.2.0

`v1.2.0` 是相册体验增强版本，重点把相册从“分类入口”推进到“可管理的相册控制台”。

## 新增能力

- 相册首页支持按名称和描述搜索。
- 相册卡片新增打开、编辑、删除三个明确操作。
- 新增相册编辑浮窗，支持修改名称和描述。
- 新增删除确认浮窗，删除相册不会删除原图片。

## 体验优化

- 相册卡片点击区域拆分，降低误触。
- 搜索无结果状态、空状态和统计信息更清晰。
- 管理动作统一使用浮窗，减少页面下拉和信息挤压。

## 验证

- `pnpm --filter @ou-image/web typecheck`
- `pnpm --filter @ou-image/api typecheck`
- `pnpm test`
