# OU-Image Hosting v1.11.0

`v1.11.0` 把账号边界、公共图库、上传、相册和日常运维串成了一套更稳、更清楚的完整体验。

## 这次更新

- 自助注册用户留在公共图床，只有站点所有者和被授权的子管理员可以进入后台。
- 后台新增用户与权限管理，可停用账号、调整子管理员权限并生成一次性密码重置链接。
- 公共图库支持最新、最热、随机与格式筛选，预览浮窗可复制链接、下载、查看原图或由管理员隐藏图片。
- 上传工作台改为先确认队列再上传，可编辑文件名、自动整理到相册和标签，并批量复制多种链接格式。
- 相册新增排序、详情管理、批量补图和自动/自定义/空白封面模式。
- S3 与 R2 配置向导补充字段说明、官方入口和可读的连接错误建议，迁移进度也更完整。
- 新增 `ouih doctor`，集中检查容器、反向代理、DNS、HTTPS、端口与服务器资源。

## 升级

```bash
ouih update
ouih doctor
```

更新流程会保留 `.env.production`。如果构建或启动失败，`ouih` 会指出失败阶段并给出恢复提示。

- 安装器与 `ouih` 脚本测试。
