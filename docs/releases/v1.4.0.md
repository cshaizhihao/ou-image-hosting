# OU-Image Hosting v1.4.0

`v1.4.0` 是概况页图形 UI 升级版本，重点让后台首页从“数据卡片列表”变成更有判断力的工作区状态面板。

## 新增能力

- 新增工作区健康状态 Hero 面板。
- 新增容量水位环，直观展示存储占用比例。
- 新增容量状态文案：健康、偏高、接近上限、未设置上限、等待第一张图片。
- 新增平均图片大小统计。
- 快捷入口扩展到上传、图片库、相册、数据统计和站点公共上传配置。

## 体验优化

- 后台概况页加宽到 1480px，更适合桌面端。
- 主要标题、说明文字和指标卡字号增大。
- 指标卡加入轻量图形反馈，减少纯文本空旷感。
- 上传快捷入口指向后台上传工作台 `/upload`。

## 验证

- `pnpm --filter @ou-image/web typecheck`
- `pnpm --filter @ou-image/api typecheck`
- `pnpm test`
