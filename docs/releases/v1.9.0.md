# OU-Image Hosting v1.9.0

`v1.9.0` 是第 10 轮最终收口版本，重点是把第 6–10 轮新增能力整理成一个可安装、可维护、文档完整的版本。

## 本轮收口

- README 更新为当前正式功能状态。
- 版本号统一提升到 `v1.9.0`。
- 发布说明补齐公共上传、登录用户历史、公开/隐藏、相册批量管理和 UI 比例优化后的最终状态。
- 保持 README 只展示一个主页面截图，不加入构建路线图或隐私提示词内容。

## 当前核心能力

- 公共图床首页可选访客直传或登录后上传。
- 登录用户可查看自己的上传历史，并批量公开/隐藏图片。
- 公共图库缩略图点击后在站内浮窗预览，支持键盘、鼠标、按钮和移动端手势。
- 图片库可多选加入多个相册；相册详情可批量移出图片。
- 安装器支持依赖自动补齐、Cloudflare 小黄云、Caddy 源站证书和 `ouih` 管理命令。
- 后台整体字号、间距和动效已针对 100% 浏览器缩放优化。
