# OU-Image Hosting v1.10.0

本轮重点打磨图片库的日常管理体验，让多选后的动作更集中、更清楚，也让每张图片的公开、收藏和相册归属状态一眼可见。

## 更新内容

- 图片库批量工具栏新增公开、隐藏、收藏、取消收藏、加入相册和移入回收站。
- 多选状态下显示已选图片数量、其中公开数量和收藏数量，减少误操作。
- 图片卡片新增状态标签：公开/私密、所属相册数量、收藏。
- 后端 `/uploads/bulk` 新增 `set-favorite` 批量动作，收藏状态不再依赖逐张更新。

## 验证

- API 批量操作回归测试覆盖公开、隐藏、收藏、取消收藏、缺少参数和移入回收站。
- 本轮使用低并发检查，避免长时间占用服务器 CPU。

## 升级建议

已有安装可通过 `ouih update` 更新。更新后刷新后台图片库页面即可看到新的批量工具栏和卡片状态标签。
