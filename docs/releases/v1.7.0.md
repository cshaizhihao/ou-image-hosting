# OU-Image Hosting v1.7.0

`v1.7.0` 完成第 8 轮：相册与图片库关联体验补强。

## 新增内容

- 相册详情页支持多选图片。
- 当前相册内可一键全选/取消全选。
- 支持批量把图片从当前相册移出，图片不会被删除。
- 后端新增 `/uploads/bulk` 的 `remove-from-albums` 操作。

## 体验优化

- 当前封面按钮有明确选中态。
- 如果被移出的图片正好是相册封面，系统会自动清空失效封面。
- 相册空状态更直接提示：去图片库多选后加入相册。

## 验证

- `pnpm --filter @ou-image/api typecheck`
- `pnpm --filter @ou-image/web typecheck`
- `pnpm exec vitest run apps/api/src/app.test.ts --maxWorkers=1 --minWorkers=1`
