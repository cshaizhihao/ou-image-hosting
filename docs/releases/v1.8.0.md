# OU-Image Hosting v1.8.0

`v1.8.0` 完成第 9 轮：整体 UI 比例、字号、动效和浮窗一致性优化。

## 调整重点

- 后台侧栏从 272px 调整到 292px，导航和工作区切换器更宽松。
- 顶部栏高度、搜索框、图标按钮和用户菜单同步放大。
- 设置中心表单、标题、标签页和输入框提升可读性。
- 相册管理里的资源列表、编辑区和图片卡片文字不再偏小。
- 全局 motion token 调整为更稳定的 140/200/260ms 节奏。

- `pnpm --filter @ou-image/web typecheck`
