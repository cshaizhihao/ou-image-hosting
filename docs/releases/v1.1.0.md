# OU-Image Hosting v1.1.0

`v1.1.0` 是一次面向产品体验的功能版本，重点完善公共图床入口、相册管理、品牌自定义和后台可读性。

## 重点变化

- 站点根地址现在是公共上传页，访客可以直接上传图片；后台上传工作台迁移到 `/upload`。
- 管理员可在「设置中心 → 站点与处理」开启或关闭公共上传、公共图库，以及访客上传时是否默认公开展示。
- 公共图库只使用缩略图展示公开图片，降低访客浏览造成的流量消耗。
- 相册页变成真正的相册主分类页，支持右上角新建相册浮窗和相册卡片入口。
- 图片库支持批量把图片加入一个或多个相册，同一张图片可以重复归类。
- 站点名称、描述、Logo、公共首页文案和登录页文案都可以在后台修改。
- 分享链接创建完成后改为浮窗展示，避免详情侧栏继续向下撑开页面。
- 后台常用文字、表单和侧栏字号调大，桌面 100% 缩放下更接近日常使用尺度。
