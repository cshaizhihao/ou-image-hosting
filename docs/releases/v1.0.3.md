# OU-Image Hosting v1.0.3

`v1.0.3` 是自动 HTTPS 与 `ouih` 管理功能的可靠性热修版本。

## 修复

- Cloudflare 边缘验证必须同时返回 HTTP 2xx 和 Cloudflare 标识，526/403 不再误报成功。
- 重复执行一键安装会保留所有未显式覆盖的生产配置，包括外部服务和自定义变量。
- 延迟启动、状态、日志和停止操作统一通过 `ouih`，不会遗漏 Caddy 的 `https` profile。

## 升级

从任何旧版本重新运行一键安装：

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```

已有 `ouih` 的安装可直接运行：

```bash
ouih update
```
