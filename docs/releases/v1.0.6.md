# OU-Image Hosting v1.0.6

`v1.0.6` 修复 Cloudflare/Caddy 部署时公网登录跳转泄露内部
`:3000` 端口的问题，并改善 `ouih` 交互菜单体验。

## HTTPS 修复

- Web 重定向优先使用生产配置中的 `APP_ORIGIN`。
- Caddy 明确转发 HTTPS、公开 Host 和 443 端口。
- 安装器完成前检查首页跳转，不允许跳到内部 Web 端口或其他域名。
- `https://your-domain/` 会保持标准 443 地址并跳转到 `/login`。

## ouih 菜单

- 查看状态、日志、地址、目录、启停和更新完成后保留结果。
- 显示“按任意键返回上级菜单”，按键后重新显示主菜单。
- 无效输入会提示并继续，不再直接退出。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```
