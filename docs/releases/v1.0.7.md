# OU-Image Hosting v1.0.7

`v1.0.7` 完成后台概况、导航稳定性、存储配置教程、桌面端可读性和动效性能优化。

## 概况与导航

- `/overview` 改为真实数据面板，显示图片数量、存储占用、容量比例、工作区和角色。
- 站点所有者可从概况页直接进入存储配置。
- 页面切换时复用已验证会话快照，管理与系统菜单不再在加载期间突然消失。
- 会话失败、退出和工作区切换仍会清理或更新快照，不扩大权限范围。

## S3 / R2 教程

- Amazon S3 与 Cloudflare R2 卡片新增独立问号入口。
- 每种提供商提供 4 步配置流程、步骤计数、上一步、下一步和完成操作。
- 支持方向键切换、Esc 关闭、焦点锁定与关闭后焦点恢复。
- 内置 AWS、Cloudflare 官方控制台和文档链接。

## 界面与动效

- 桌面侧栏、顶栏、导航字号和常用控件整体提高约 8–12%。
- 主内容最大宽度提高到 1480px，各业务页同步放宽，减少宽屏无效留白。
- 移除高频卡片漂浮、图片缩放和按钮按压缩放。
- 抽屉、弹窗和菜单动画缩短，并保留 `prefers-reduced-motion` 支持。
- 高频进度条使用 `transform: scaleX()`，避免连续更新 `width` 触发布局重排。

## 验证

- `pnpm check` 全部通过。
- 52 个单元测试通过。
- 安装器与 `ouih` 集成测试通过。
- Playwright 7/7 通过，包含 Axe、核心流程、响应式和减少动画检查。

## 更新

```bash
ouih update
```

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```
