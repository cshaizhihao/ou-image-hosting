# OU-Image Hosting v1.6.0

`v1.6.0` 完成第 7 轮：登录用户历史上传与公开/隐藏批量管理。

## 新增内容

- 公共上传页新增“我的上传历史”模块。
- 登录用户可以看到自己账号下最近上传的图片。
- 支持多选历史图片后批量“设为公开”或“设为隐藏”。
- 后端新增 `/uploads/bulk` 的 `set-public-visibility` 操作，统一处理公开状态。

## 体验优化

- 上传完成后自动刷新“我的上传历史”。
- 公开/隐藏完成后自动刷新公共图库。
- 历史上传卡片显示“正在公共图床展示 / 仅自己可见”，状态更直接。

- `pnpm --filter @ou-image/web typecheck`
