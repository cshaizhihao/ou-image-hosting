# OU-Image Hosting v1.1.1

`v1.1.1` 是安装与更新体验补丁，重点解决精简系统执行 `ouih update` 时缺少 Git 导致更新失败的问题。

## 修复内容

- `ouih update` 现在会自动检测并补齐 Git、curl、OpenSSL、coreutils、CA 证书。
- 保持更新流程的生产配置保护：更新前备份 `.env.production`，同步代码后恢复原配置和密钥。
- 清理 `ouih` 的运行环境检查逻辑，避免重复 Docker 状态判断。

- `bash -n install.sh`
