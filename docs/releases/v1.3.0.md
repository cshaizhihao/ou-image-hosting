# OU-Image Hosting v1.3.0

`v1.3.0` 是公共上传体验增强版本，重点让访客上传更接近真正可用的公共图床入口。

## 新增能力

- 支持多图选择和多图拖拽上传。
- 支持直接粘贴剪贴板图片上传。
- 上传按钮显示批量进度，例如 `正在上传 2/5`。
- 新增最近上传列表，保留最近 8 条上传结果。
- 最近上传支持一键复制全部原图链接或全部 Markdown。

## 体验优化

- 公共上传卡片文案更明确，提示拖拽、批量选择和粘贴三种方式。
- 单图结果与批量历史并存，访客连续上传后可以集中复制链接。
- 最近上传卡片使用轻量浮层样式，不挤压主上传区域。

## 验证

- `pnpm --filter @ou-image/web typecheck`
- `pnpm --filter @ou-image/api typecheck`
- `pnpm test`
