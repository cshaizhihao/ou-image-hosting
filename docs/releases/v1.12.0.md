# OU-Image Hosting v1.12.0

`v1.12.0` 继续把“好看、顺手、稳定”往前推了一大步：公共上传更安全，大图库更稳，品牌可自定义，备份与运维也更适合长期自托管。

## 这次更新

- 公共上传加入匿名/登录用户分离的速率、每日数量和每日流量限制，并提供可选短时效人机验证与 IP 封禁。
- 图片库和公共图库加入分页偏好、稳定页码、懒加载缩略图和局部状态更新，大图库滚动更顺。
- 公共图库在签名链接开启时不再公开缓存，避免用户拿到已经过期的图片地址。
- 后台、登录页和公共上传页统一读取站点品牌配置，可调整站点名称、副标题、Logo、文案、主题模式和受控主题色。
- 远程 URL 上传加强网络边界，阻断私网、保留地址、IPv4-mapped IPv6 和 DNS rebinding 风险。
- 存储中心新增应用内备份预检、自动备份调度和更清晰的恢复确认。
- `ouih` 增加备份、恢复、回滚、备份目录检查、版本一致性检查和失败恢复提示。

## 升级

```bash
ouih update
ouih doctor
```

如果你使用 Cloudflare 小黄云或自建反向代理，升级后建议打开后台公共页与图片库各点一次，确认公开地址没有暴露内部端口。
