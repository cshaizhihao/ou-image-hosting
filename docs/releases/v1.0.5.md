# OU-Image Hosting v1.0.5

`v1.0.5` 修复一行安装命令通过标准输入执行时的启动错误。

## 修复

- `curl ... | bash` 不再因 `BASH_SOURCE[0]` 未定义而退出。
- 保持直接运行 `bash install.sh` 和测试中 `source install.sh` 的兼容性。
- 新增标准输入执行回归测试，防止同类问题再次出现。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```
