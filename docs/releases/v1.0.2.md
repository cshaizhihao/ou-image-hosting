# OU-Image Hosting v1.0.2

`v1.0.2` 让公网安装从“应用容器启动”升级为“域名 HTTPS 真正可访问”。

## 主要更新

- HTTPS 域名默认部署 Caddy，自动申请与续期证书。
- 安装器预检 80/443，完成后验证 TLS 与反向代理链路。
- 新增全局 `ouih` 管理命令和交互菜单。
- 支持 Cloudflare 小黄云 + `Full (strict)` 安装，并由 Caddy 通过 ACME 自动维护受信源站证书。
- 支持安全更新、状态与日志查看、启停及保留数据的卸载。
- 保留 `external` 模式，兼容已有 Nginx、Caddy 或服务器面板。

## 升级

已有安装直接重新执行：

```bash
curl -fsSL https://raw.githubusercontent.com/cshaizhihao/ou-image-hosting/main/install.sh | bash
```

若原来配置的是 HTTPS 域名，安装器会默认启用内置 Caddy。请确保域名 A/AAAA 记录指向服务器，并开放 TCP 80/443。

升级完成后可使用：

```bash
ouih
ouih url
ouih status
```
